# Example API

This is an example of a subgraph to the Infratographer super graph. It is a simple API that returns a list of example components. It is intended to be used as a template for creating new subgraphs.


## Example Components

```mermaid
graph LR;
    Example-->Name;
    Example-->Description;
 ```
